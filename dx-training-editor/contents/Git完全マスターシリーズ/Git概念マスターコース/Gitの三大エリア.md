# Gitの三大エリア

Gitには3つのエリアがあります。

1. **作業ツリー** — 実際にファイルを編集する場所
2. **ステージングエリア** — コミットする変更を選ぶ待合室
3. **リポジトリ** — 確定した変更が保存される場所

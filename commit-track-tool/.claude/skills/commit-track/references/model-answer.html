uvを使えばpython仮想環境を立ち上げる手間をなくせます。
# 先週 Anaconda から uv へ移行（PR #17）したので、データ取得はこれで揃える
$ cd commit-track-tool
$ uv sync
$ uv run python src/comitora_data_collector.py --owner ug-kitamura --repo AI_Driven_School
# 生成物は output/report_data.json → スキルで HTML に（このレポートと同じ流れ）

ルートに pyproject.toml / uv.lock があるので、GitHub Actions の実行ステップも uv sync → uv run … に寄せると、ローカル検証と本番の差分が減ります。

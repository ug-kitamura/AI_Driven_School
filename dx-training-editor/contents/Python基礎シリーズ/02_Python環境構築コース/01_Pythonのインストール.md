# Pythonのインストール

## Windows

1. [python.org](https://www.python.org/) からインストーラーを取得
2. インストール時に **「Add python.exe to PATH」** にチェックを入れる
3. コマンドプロンプトで確認:

```bash
python --version
```

## Mac

```bash
brew install python
python3 --version
```

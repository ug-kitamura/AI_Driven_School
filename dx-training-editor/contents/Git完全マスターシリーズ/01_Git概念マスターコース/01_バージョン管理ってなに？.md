# バージョン管理ってなに？

バージョン管理とは、ファイルの変更履歴を記録・管理するしくみです。

## なぜ必要？

- 過去のバージョンに戻せる
- 誰がいつ何を変更したか分かる
- 複数人で同じファイルを編集できる

### 確認問題

バージョン管理の主な目的は何ですか？

- [ ] ファイルを圧縮する
- [x] 変更履歴を記録する
- [ ] ファイルを暗号化する

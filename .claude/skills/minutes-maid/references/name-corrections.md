# 固有名詞補正リスト

音声認識の誤変換が多い名詞を管理します。文字起こしを読む際にこのリストを適用し、誤った表記を正しい表記に置き換えてください。

## 使い方

- **左列**（誤）：音声認識で間違えやすい表記
- **右列**（正）：正しい表記
- 部分一致で適用してよい（例：「木村」→「北村」は「木村さん」「木村が」なども対象）

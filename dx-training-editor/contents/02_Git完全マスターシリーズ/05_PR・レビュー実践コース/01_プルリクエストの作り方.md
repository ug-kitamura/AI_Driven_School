# プルリクエストの作り方

PR（プルリクエスト）は変更内容をレビューしてもらうための仕組みです。

## 手順

1. フィーチャーブランチを作成してコミット
2. GitHub にプッシュ
3. GitHub の画面で「Compare & pull request」をクリック
4. タイトルと説明を書いてレビュアーを指定

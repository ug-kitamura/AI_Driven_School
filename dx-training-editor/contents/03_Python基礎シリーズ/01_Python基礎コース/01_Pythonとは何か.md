# Pythonとは何か

このレッスンを終えると、次のことができるようになります。

- Python がどのようなプログラミング言語かを説明できる

---

## 1. プログラミング言語とは？

コンピューターは、私たちが話す日本語や英語をそのままは理解できません。  
そこで、**コンピューターに命令を伝えるための専用の言語**が必要になります。それが「プログラミング言語」です。

<!-- 図解イメージ：人間 → プログラミング言語 → コンピューター の矢印図 -->

世界にはたくさんのプログラミング言語があります（Java、C、JavaScript など）。  
その中でも、**Python（パイソン）** は特に「シンプルで読みやすい」ことで人気の高い言語のひとつです。

---

## 2. Python ってどんな言語？

<!-- 画像プロンプト：Pythonのロゴマークと「シンプル・パワフル」をイメージしたイラスト -->
![Pythonの5つの特徴を放射状バッジで囲んだインフォグラフィック](images/python-features-radial-infographic.png)

Python は **1991年** にグイド・ヴァン・ロッサムによって開発されました。  
名前の由来はヘビではなく、イギリスのコメディ番組「**モンティ・パイソン**」からきています。

### Python の主な特徴

| 特徴 | 説明 |
|------|------|
| **読みやすいコード** | 英語に近い自然な文法で書ける |
| **少ない記述量** | 他の言語より短いコードで同じ処理が書ける |
| **豊富なライブラリ** | 便利な道具が最初から多数用意されている |
| **幅広い用途** | Web・AI・データ分析など様々な分野で使える |
| **大きなコミュニティ** | 世界中に利用者がおり、情報が豊富 |

---

## 3. Python で何ができる？

Python は非常に幅広い用途で活躍しています。

### 🤖 人工知能・機械学習
ChatGPT のような AI の開発や、データから予測モデルを作る分野で最もよく使われる言語です。

### 📊 データ分析・自動化
Excel の代わりに大量のデータを処理したり、毎日行う繰り返し作業を自動化したりできます。

### 🌐 Web 開発
Instagram や Pinterest のバックエンド（裏側の仕組み）も Python で作られています。

### 🔬 科学・研究
科学計算や統計処理にも広く使われており、大学・研究機関でも人気です。

## 4. はじめての Python コード

百聞は一見にしかず。Python のコードを少しだけ見てみましょう。

### 画面に文字を表示する

```python
print("こんにちは、Python！")
```

**出力結果：**
```
こんにちは、Python！
```

### 計算をする

```python
print(1 + 2)
print(10 * 5)
```

**出力結果：**
```
3
50
```

どうでしょう？とてもシンプルに書けることがわかりますね。  
`print()` は「（）の中身を画面に表示する」という命令です。次のレッスン以降で詳しく学んでいきます。

## 5. Python を学ぶとどんないいことがある？

- ✅ **求人が多い** — AI・データ活用の需要増加により、Python スキルの求人は年々増えています
- ✅ **初心者にやさしい** — 文法がシンプルなため、最初の一歩が踏み出しやすい
- ✅ **すぐに成果が出せる** — 業務の自動化など、学んだ内容をすぐ実務に活かせる
- ✅ **広がりがある** — Web・AI・データ分析など、興味に合わせてスキルを伸ばせる

# **私たちの戦略 (Our Strategies)**

## **#1: Individual focus（個人にフォーカス）**

### **どのようにパーパスを実現するのか？**
「個人」を重視し、私たちは以下の項目(1)～(5)に取り組むことで、「各個人の物事を成し遂げる能力と解決策を導き出す力を高めること」を目指します。これにより、ABC/XYZのパーパス実現へとつなげます。
なお、設計開発エリア内の「個人」を象徴するメタファーとして**花**のロゴを使用しています。

| No. | コンセプト | 説明 |
|:---:|:---|:---|
| (1) | **Soil（土壌）** | 個人の成長基盤であり、「PMT（Process, Method, Tool）の情報源」を表しています。私たちの日ごろの仕事の源泉となる部分です。 |
| (2) | **Plant Root（根）** | 個人が成長するためにはPMT情報が必要です。ここでは「花（＝個人）への水（＝PMT情報）の届きやすさ（効率）」、すなわちPMT情報と個人との間のインターフェースを意味しています。 |
| (3) | **Sunshine Management（太陽の光）** | 「トレーニングを通して、知識として存在するPMT情報を、実際に活用して使える状態すること」を光合成としており、それをSunshine managementで表現しています。 |
| (4) | **Pouring Water（水やり）** | 「一人ひとりの状態に合わせた方法で成長を促し、気づきを与え、PMT情報を単に使えるだけでなく、より効果的かつ効率的に活用できるためのコーチング」を、Pouring waterで表現しています。 |
| (5) | **Leaf（葉）** | 「トレーニングやコーチングで得た知識を積極的に取り入れ、活用し、実践していこうとする個人の意志」です。これは光合成によってエネルギーを生み出す主役である「葉」そのものです。私たちは、そこにポジティブな働きかけができるようサポートしていきます。 |

## **#2: Environment focus（環境にフォーカス）**

### **どのようにパーパスを実現するのか？**
ここでは「環境」に焦点を当てます。項目(6)～(10)への改善・貢献を通じて、「各個人が効果的かつ効率的に動ける環境を整えること」を目指します。これにより、ABC/XYZのパーパス実現へとつなげます。

| No. | コンセプト | 説明 |
|:---:|:---|:---|
| (6) | **Maintain right content PMT（適切なコンテンツのPMTを維持する）** | PMT情報が常に正確かつ意味のあるデータであり続けるよう、その維持に適した環境を整備します。 |
| (7) | **Keep healthy Efficiency（健全な効率性を保つ）** | コンプライアンスを伴って創出される「健全な効率性」を重視し、組織全体の効率性を損なわないための環境作りに努めます。 |
| (8) | **Future-proof（将来にわたって通用するようにする）** | 将来の不確実性（ビジネスニーズ、市場トレンドなど）に対する予防的な活動に注力し、早期に準備できる環境を整えます。 |
| (9) | **We are One ABC（私たちは一つのABCである）** | すべてのABCプロダクト領域において、組織間の風通しを促進し、効率性を水平展開できる環境を整備します。 |
| (10)| **We belong to EA（私たちはEAの一員である）** | 公平性、透明性、協調性に基づき、積極的に情報共有を行い、優れた取り組みをEA内で水平展開できる環境作りを目指します。 |

**用語補足**
- `ABC` は部門の名前を表します
- `ABC/XYZ` は私たちの部の名前を表します
- `PMT` は Process, Method, Tool の略です
- `EA` は East Asia の略です

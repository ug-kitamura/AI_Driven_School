# Anaconda セットアップ

## 学習目標

このレッスンを終えると、以下ができるようになります。

- Anaconda が何であるかを説明できる
- 社内手順に従って Anaconda をインストールできる
- インストールが正常に完了したか確認できる
- 仮想環境を作成・有効化できる

---

## Anaconda とは

**Anaconda** は、Python 本体に加えて、データ分析や機械学習でよく使われるライブラリ（NumPy・pandas・Jupyter Notebook など）をまとめてインストールできるディストリビューションです。

個別にライブラリをインストールする手間が省けるため、Python を始めて使う方に広く利用されています。

## インストール前の確認事項

> ⚠️ **公式サイトからの直接ダウンロードは不可です。**
> 他のツール（Git・VS Code など）と同様に、社内の申請手順を経てインストールしてください。
> 申請からインストール完了まで数日かかる場合があるため、**研修開始の 1 週間前には申請を出すことを推奨します。**

申請前に以下を確認しておきましょう。

- [ ] 所属部署の IT ポリシーで Anaconda の使用が許可されているか
- [ ] すでに Python がインストールされていないか（競合が起きる場合があります）
- [ ] ディスクの空き容量が **5 GB 以上** あるか

---

## インストール手順

### 1. 社内申請

IT ヘルプデスクのポータルから「開発ツール導入申請」を行います。申請フォームの「ツール名」欄に **Anaconda** と記入してください。

<!-- 社内申請ポータルのスクリーンショット（申請フォーム記入例） -->

### 2. インストーラーの入手

申請承認後、IT ヘルプデスクから共有されるリンク、または社内共有ドライブからインストーラーをダウンロードします。

> ℹ️ インターネット上の Anaconda 公式サイト（anaconda.com）から直接ダウンロードしないでください。社内で動作確認済みのバージョンが配布されています。

### 3. インストーラーの実行

#### Windows の場合

1. ダウンロードした `.exe` ファイルをダブルクリックする
2. 「Next」→「I Agree」をクリックする
3. インストール対象を **「Just Me」** に設定する（推奨）
4. インストール先フォルダを確認し「Next」をクリックする
5. 以下のオプションを確認する
   - **「Add Anaconda to my PATH environment variable」** → **チェックしない**（推奨）
   - **「Register Anaconda as my default Python」** → チェックする
6. 「Install」をクリックしてインストールを開始する
7. 完了後「Finish」をクリックする

<!-- Windows インストーラーのオプション画面スクリーンショット -->

#### macOS の場合

1. ダウンロードした `.pkg` ファイルをダブルクリックする
2. 画面の指示に従い「続ける」→「同意する」をクリックする
3. インストール先を確認し「インストール」をクリックする
4. 完了後「閉じる」をクリックする

## インストールの確認

インストールが完了したら、正しく動作しているか確認しましょう。

### Anaconda Navigator の起動

スタートメニュー（Windows）または Launchpad（macOS）から **Anaconda Navigator** を起動します。以下のような画面が表示されれば成功です。

<!-- Anaconda Navigator の起動画面スクリーンショット -->

### コマンドで確認する

**Anaconda Prompt**（Windows）またはターミナル（macOS）を開き、次のコマンドを実行してください。

```bash
conda --version
```

以下のようにバージョン番号が表示されれば正常にインストールされています。

```
conda 23.x.x
```

続いて Python のバージョンも確認します。

```bash
python --version
```

```
Python 3.x.x
```

## 仮想環境の作成と有効化

Anaconda では **仮想環境** を使って、プロジェクトごとに異なるパッケージ構成を管理できます。

### 仮想環境を作成する

```bash
conda create -n myenv python=3.11
```

- `myenv` は環境名です。自由に変更できます。
- `python=3.11` でインストールする Python のバージョンを指定します。

途中で `Proceed ([y]/n)?` と聞かれたら `y` を入力してください。

### 仮想環境を有効化する

```bash
conda activate myenv
```

プロンプトの先頭に `(myenv)` と表示されれば、仮想環境が有効になっています。

```
(myenv) C:\Users\yourname>
```

### 仮想環境を無効化する

```bash
conda deactivate
```

レッスンの終わりに、以下をセルフチェックしましょう。

- [ ] `conda --version` でバージョンが表示される

## トラブルシューティング

| 症状 | 対処法 |
|---|---|
| `conda` コマンドが見つからない | Anaconda Prompt を使用しているか確認する。通常のコマンドプロンプトでは動作しない場合がある |
| インストーラーが起動しない | IT ヘルプデスクに問い合わせる |
| 既存の Python と競合している | IT ヘルプデスクに相談し、既存 Python の扱いを確認する |

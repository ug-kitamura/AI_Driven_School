git log --follow
ファイル名変更をまたいでコミット履歴を追跡
# 今期は多くのファイルがリネーム・移動された
$ git log --follow --oneline commit-track-tool/src/comitora_data_collector.py

5670891d update comitora script
db5715f3 update python scripts
# --follow なしでは rename 前の履歴が途切れる
$ git log --follow -p -- commit-track-tool/src/comitora_report_generator.py

ファイルのリネームや移動が多いリポジトリでは --follow オプションが必須です。これにより rename 前のコミットまで遡って差分を確認できます。今期のような大規模リファクタリング後に特に役立ちます！

# GitHubアカウントの作成

## 学習目標

このレッスンを終えると、以下ができるようになります。

- GitHub の公式サイトでアカウントを新規作成できる
- 基本的なプロフィール設定を完了できる
- メール認証を通過し、GitHub を利用可能な状態にできる

---

## 事前準備

- 有効なメールアドレス（未登録のもの）
- 使用するブラウザ（Chrome / Firefox / Edge など最新版推奨）

---

## 手順

### 1. GitHub 公式サイトにアクセスする

ブラウザで以下の URL を開きます。

```
https://github.com
```

<!-- スクリーンショット: GitHub トップページ（Sign up ボタンが見える状態） -->

---

### 2. サインアップを開始する

トップページ右上の **「Sign up」** ボタンをクリックします。

<!-- スクリーンショット: Sign up ボタンのクローズアップ -->

---

### 3. アカウント情報を入力する

以下の情報を順に入力します。

| 項目 | 内容 |
|------|------|
| メールアドレス | 事前準備で用意したアドレス |
| パスワード | 15 文字以上、または数字・小文字を含む 8 文字以上 |
| ユーザー名 | 英数字・ハイフンのみ使用可。後から変更可能 |

> **ユーザー名のポイント**  
> ユーザー名は公開 URL（例: `https://github.com/yourname`）に使われます。  
> チーム内で識別しやすい名前にしておくと便利です。

<!-- スクリーンショット: サインアップフォームの入力例 -->

---

### 4. ロボット認証を通過する

画面の指示に従い、パズル形式の認証（CAPTCHA）を完了します。

<!-- スクリーンショット: CAPTCHA 画面の例 -->

---

### 5. メール認証を行う

登録したメールアドレスに GitHub から認証コードが届きます。

1. メールを開き、**6 桁の認証コード** を確認する
2. GitHub の画面にコードを入力する

<!-- スクリーンショット: メール認証コード入力画面 -->

> メールが届かない場合は、迷惑メールフォルダを確認してください。  
> それでも届かない場合は **「Resend code」** をクリックして再送できます。

---

### 6. プラン選択

**「Free」** プランが選択されていることを確認し、そのまま進みます。  
（個人利用・オープンソース開発には Free プランで十分です）

<!-- スクリーンショット: プラン選択画面 -->

---

### 7. アカウント作成完了

ダッシュボード画面が表示されればアカウント作成は完了です。

<!-- スクリーンショット: GitHub ダッシュボード（初回ログイン直後） -->

---

## 確認ポイント

- [ ] GitHub にログインできる
- [ ] ダッシュボードが表示される
- [ ] プロフィールページ（`https://github.com/ユーザー名`）にアクセスできる

---

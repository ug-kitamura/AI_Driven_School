お役立ち情報

💡 GitHub Actions の GITHUB_TOKEN を活用しよう

今週 PR #37 で GH_TOKEN → GITHUB_TOKEN に統一されました。GitHub Actions では、何も設定しなくても GITHUB_TOKEN が自動で発行されるため、PAT を作らなくても自リポジトリへの API アクセスが可能です。

jobs:
  report:
    permissions:
      contents: read
      issues: read
      pull-requests: read
    steps:
      - uses: actions/checkout@v4
      - name: Run comitora
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
        run: uv run python src/main.py \
          --owner ${{ github.repository_owner }} \
          --repo ${{ github.event.repository.name }}

permissions ブロックで必要最小権限を明示すると、よりセキュアなワークフローになります。

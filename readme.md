# AI Driven School

## ツール一覧

### [DX Training Studio](dx-training-studio)

DX ツールトレーニングのコンテンツ計画・作成・編集・デプロイを支援する統合スタジオ（4ペイン UI）。

### [Commit & Track tool（コミとら）](commit-track-tool)

GitHubのコミット履歴をAIで分析し、図解レポートをSlackに自動投稿するツール。

### [Minutes Maid（ミニッツメイド）](minutes-maid)

会議の音声文字起こしファイル（.vtt）を元に、議事録html図解を作成するツール。
議事録 `minutes ミニッツ` を作るので Minutes Maid
